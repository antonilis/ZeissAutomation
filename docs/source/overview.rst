Overview
========

ZeissAutomation is a Python project for automating confocal microscopy workflows (e.g., Zeiss microscopes).

This documentation describes the modules included in this repository and how to use them.

Modules in this project:

- IO — input/output helpers for experiment files
- ZeissAPI — integration with microscope control and Zen Blue
- config — project configuration
- data_preprocessing — data analysis pipeline components
- utils — utility functions used across modules
